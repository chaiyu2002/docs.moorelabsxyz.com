# 07-03 Lyquor and Open Financial Infrastructure

Related blog post: `blog/2026-07-03-lyquor-open-financial-infrastructure/index.md`

## English

On Jul 3, we clarified a sharper distinction between HyperCore and Lyquor: the question is not only how financial applications execute, but who gets to define the financial infrastructure. HyperCore provides specialized trading infrastructure as part of Hyperliquid's core system. Lyquor exposes the capability to build that kind of infrastructure as sequenced Lyquid network applications, where matching, clearing, risk, margin, liquidation, and settlement logic can be built, inspected, and composed by developers.

## Chinese

7 月 3 日，我们进一步明确了 HyperCore 和 Lyquor 的区别：关键问题不只是金融应用如何执行，而是谁来定义金融基础设施。HyperCore 把专用交易基建作为 Hyperliquid 核心系统的一部分来提供。Lyquor 则开放了构建这类基建的能力，让撮合、清算、风控、保证金、强平和结算逻辑可以作为被排序驱动的 Lyquid network applications，由开发者构建、检查和组合。
