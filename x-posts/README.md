# X Posts

This directory stores short posts prepared for publication on X.

Use one Markdown file per post or thread. Prefer date-based filenames that match
the related blog post when there is a direct source article.

Recommended filename pattern:

```text
YYYY-MM-DD-topic.md
```

Each file should include:

- Related blog post link or path
- English X post
- Chinese X post, when needed
- Notes about publication status or final edits
